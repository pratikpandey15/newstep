# MOSFET Basics

   ## What is MOSFET?
   MOSFET stands for Metal Oxide Semiconductor Field Effect Transistor.
   It is a voltage controlled device used for amplification and switching.

   ## Types of MOSFET
   - **N-Channel MOSFET** — electrons are majority carriers
   - **P-Channel MOSFET** — holes are majority carriers

   ## Three Operating Regions
   | Region | Condition | Behaviour |
   |--------|-----------|-----------|
   | Cutoff | VGS < Vth | No current flows |
   | Linear | VDS < VGS - Vth | Acts like resistor |
   | Saturation | VDS > VGS - Vth | Acts like current source |

   ## Key Parameters
   - **Vth** — Threshold Voltage
   - **gm** — Transconductance
   - **kn** — Process Transconductance
   - **W/L** — Width to Length Ratio
